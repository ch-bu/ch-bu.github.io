Standards

Designed by Freepik
Standards kennt jeder. Ein USB-Kabel funktioniert an jedem Laptop, da sich die Hersteller an die Regeln halten. Nicht anders ist es bei E-Learning Produkten. Als es nur Stift und Papier gab, war das kein Problem. Portabilität ging ohne Probleme. Auch beim Tageslichtprojektor. Mit der Technologisierung hat sich das allerdings geändert.

SCORM (Sharable Content Object Reference Model)

SCORM tells programmers how to write their code so that it can “play well” with other e-learning software. It is the de facto industry standard for e-learning interoperability.

Verwaltet wird SCORM durch ADL (Advanced Distributed Learning)

SCORM besteht aus 4 Büchern.

What is SCORM?
SCORM ist der bekannteste Standard im E-Learning. Es ist aber nicht der einzige. Schlussendlich ermöglicht SCORM die Portabilität von E-Learning Modulen zwischen verschiedenen LMS. Nicht alle Anbieter benutzen allerdings SCORM. Z.B. Udacity, edx.

Tin Can API / Experience API / x-API

Lernen findet überall statt. Die Standards sollten dieser Tatsache gerecht werden.

xAPI speichert Lernverhalten anhand von Statements: "Hannes las Kapitel 2 aus Buch X".

Es ist nichts anderes als ein Web API

Tin Can API, tincanapi.com

Was x-APi speichern kann

  • Training data from a simulation
  • Watching a training video, stopping and starting it
  • Reading an article or interacting with an eBook
  • Physiological measures, such as heart-rate data
  • Quiz scores and answer history by question

Euer E-Learning Modul

In dieser Aufgabe macht ihr euch erste Gedanken zu eurem E-Learning Modul. Am Ende der Aufgabe solltet Ihr eine erste Fassung eures Moduls auf Papier aufgezeichnet haben.

Überlegt euch zunächst einzeln, was der Inhalt und das Ziel eurer Lernumgebung ist (ihr könnt es noch ändern, ihr könnt auch mehrere Ideen sammeln). Schreibt eure Ideen auf (15 Minuten).

Tauscht euch in der Dreiergruppe zu euren Ideen aus. Fragt euch als Gruppe: Ist das Modul realisierbar? Ist das Ziel erreichbar? Habt ihr erste Ideen, wie das Modul aussehen könnte? (25 Minuten)
